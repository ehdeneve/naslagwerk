2.1 Type selector:

Dit is een titel (heading 3) met een custom CSS selector

Omdat we de stijl toepassen op het type element, wordt de styling toegepast worden op alle elementen op de pagina (van dat type)

2.2 Class selector:

Heading 3 heeft nog steeds dezelfde stijl

Door het toevoegen van het class attribuut krijgt dit artikel een gele achtergrondkleur

Twee classen op 1 element

Door het toevoegen van twee class attributen krijgt dit artikel een gele achtergrondkleur uit de class 'highlighted' en de rode tekst door de class 'article'

2.3 ID selector:

Heading 3 heeft nog steeds dezelfde stijl

Door een ID selector toe te wijzen, kunnen we rechtstreeks het ID attribuut uit het element aanspreken. Aangezien elk element max. 1 ID kan hebben is dit de meest specifieke vorm van styling.
